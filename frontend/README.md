# Automated Trading System - Frontend

This directory contains the cross-platform frontend application built with Flutter.
It is designed to be highly responsive, updating 60 frames per second using WebSocket streams.

## Features to be Implemented:
1. **GPU-Accelerated Charting** (using `candlesticks` or similar Flutter charting library).
2. **Real-time Order Book** & Tape via WebSockets to the API Gateway.
3. **Portfolio Dashboard** showing Realized/Unrealized P&L.
4. **Strategy Controls** (Enable/Disable algorithms, override risk parameters).

## Setup
*To be initialized with `flutter create .`*
